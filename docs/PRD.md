# Product Requirements Document (PRD)
# AI 기반 중국어 학습 프로그램

## 1. 제품 개요

### 1.1 제품 비전
초보자가 AI 튜터의 도움을 받아 효과적으로 중국어 회화와 문법을 학습할 수 있는 통합 학습 플랫폼

### 1.2 목표 사용자
- 중국어 초보 학습자 (HSK 1-2급 수준)
- 자기주도 학습을 선호하는 성인 학습자
- 회화 실력 향상을 원하는 학습자
- 체계적인 학습 관리가 필요한 학습자

### 1.3 핵심 가치 제안
- **개인화된 학습**: AI가 사용자의 수준과 약점을 분석하여 맞춤형 학습 제공
- **실전 회화 중심**: 실생활에서 바로 사용할 수 있는 회화 표현 학습
- **과학적 복습 시스템**: 간격 반복(Spaced Repetition) 알고리즘 기반 효율적 복습
- **게임화 요소**: 레벨, 업적, 연속 학습 등으로 동기부여
- **발음 교정**: 음성 인식 및 성조 분석으로 정확한 발음 연습

## 2. 기능 요구사항

### 2.1 필수 기능 (MVP)

#### 2.1.1 단어 학습
**기능 설명**: HSK 1-2급 기반 단어 학습
- 단어장 제공 (간체자, 번체자, 병음, 의미)
- 음성 재생 기능
- 레슨별 10-15개 단어 구성
- 학습 진도 추적

**사용자 스토리**:
```
AS A 초보 학습자
I WANT TO 레벨별로 정리된 단어를 학습하고
SO THAT 체계적으로 어휘력을 쌓을 수 있다
```

**수용 기준**:
- [ ] 최소 500개 이상의 HSK 1-2급 단어 제공
- [ ] 각 단어에 대한 TTS 음성 제공
- [ ] 단어별 학습 상태 저장 (new, learning, mastered)
- [ ] 레슨별 진도율 표시

#### 2.1.2 회화 연습
**기능 설명**: 실생활 상황별 대화 학습
- 상황별 대화문 제공
- AI 튜터와 1:1 대화 연습
- 문법 오류 자동 교정
- 자연스러운 표현 제안

**사용자 스토리**:
```
AS A 회화 학습자
I WANT TO AI 튜터와 실시간으로 대화하며 연습하고
SO THAT 실전 회화 능력을 향상시킬 수 있다
```

**수용 기준**:
- [ ] 최소 10가지 실생활 상황 대화 시나리오
- [ ] Claude API 기반 자연스러운 대화 생성
- [ ] 사용자 입력에 대한 문법 교정 및 피드백
- [ ] 대화 히스토리 저장

#### 2.1.3 발음 평가
**기능 설명**: 음성 인식 기반 발음 평가
- 마이크 녹음 기능
- 음성 인식 및 텍스트 변환
- 정확도 점수 산출
- 발음 개선 피드백

#### 2.1.4 퀴즈
**기능 설명**: 학습 내용 확인 퀴즈
- 객관식 문제 생성
- 번역 문제
- 듣기 문제
- 즉시 채점 및 피드백

#### 2.1.5 학습 진도 관리
**기능 설명**: 학습 기록 및 통계
- 학습 시간 추적
- 단어 마스터 현황
- 평균 점수 계산
- 학습 곡선 시각화

### 2.2 고급 기능 (Phase 2)

#### 2.2.1 간격 반복 학습 (SRS)
**기능 설명**: SM-2 알고리즘 기반 복습 시스템
- 복습 간격 자동 계산
- 망각 곡선 기반 최적 타이밍
- 단어별 난이도 조정
- 복습 우선순위 관리

#### 2.2.2 성조 분석
**기능 설명**: 중국어 4성조 정확도 분석
- F0 주파수 추출
- 성조 패턴 매칭
- 시각적 피드백
- 원어민 발음과 비교

#### 2.2.3 게임화 시스템
**기능 설명**: 학습 동기 부여 요소
- 레벨 및 XP 시스템
- 업적 시스템 (20+ 업적)
- 연속 학습 스트릭
- 리더보드 (향후 멀티유저)

#### 2.2.4 AI 기반 개인화
**기능 설명**: 사용자 맞춤형 학습 경로
- 약점 분석 및 진단
- 맞춤형 연습문제 생성
- 학습 난이도 자동 조정
- 추천 학습 콘텐츠

#### 2.2.5 웹 인터페이스 (PWA)
**기능 설명**: 크로스 플랫폼 웹 앱
- 반응형 디자인
- 오프라인 지원
- 모바일 최적화
- 설치 가능한 PWA

## 3. 비기능 요구사항

### 3.1 성능
- **응답 시간**: AI 응답 < 3초
- **음성 인식**: 처리 시간 < 2초
- **페이지 로드**: 초기 로드 < 2초
- **데이터베이스**: 쿼리 < 100ms

### 3.2 확장성
- 사용자당 독립적인 데이터베이스
- 최소 10,000개 단어 지원
- 1년치 학습 기록 저장

### 3.3 사용성
- 직관적인 UI/UX
- 최소 클릭으로 주요 기능 접근
- 명확한 진행 상태 표시
- 도움말 및 튜토리얼

### 3.4 호환성
- **OS**: Windows 10+, macOS 10.15+, Linux (Ubuntu 20.04+)
- **Python**: 3.8+
- **브라우저**: Chrome 90+, Firefox 88+, Safari 14+

## 4. 데이터 모델

### 4.1 데이터베이스 스키마

```sql
-- 학습 세션
CREATE TABLE sessions (
    id INTEGER PRIMARY KEY,
    start_time TIMESTAMP,
    end_time TIMESTAMP,
    lesson_number INTEGER,
    words_learned INTEGER,
    quiz_score REAL
);

-- 단어 마스터 현황
CREATE TABLE word_mastery (
    word_id INTEGER PRIMARY KEY,
    simplified TEXT,
    traditional TEXT,
    pinyin TEXT,
    times_practiced INTEGER,
    times_correct INTEGER,
    last_practiced TIMESTAMP,
    mastery_level TEXT,
    next_review TIMESTAMP,
    easiness_factor REAL,
    interval INTEGER,
    repetitions INTEGER
);

-- 발음 평가 기록
CREATE TABLE pronunciation_history (
    id INTEGER PRIMARY KEY,
    word TEXT,
    score INTEGER,
    attempt_time TIMESTAMP,
    recognized_text TEXT
);
```

## 5. 개발 로드맵

### Phase 1: MVP (4주)
- Week 1: 데이터 파싱 및 단어 학습 기능
- Week 2: 회화 연습 및 AI 튜터 통합
- Week 3: 발음 평가 및 퀴즈 기능
- Week 4: 진도 관리 및 데이터베이스

### Phase 2: 고급 기능 (6주)
- Week 5-6: 간격 반복 학습 시스템
- Week 7-8: 성조 분석 및 고급 음성 기능
- Week 9-10: 게임화 시스템
- Week 11: 웹 인터페이스 (Streamlit)
- Week 12: 테스트 및 버그 수정

## 6. 성공 지표

### 6.1 학습 효과
- 4주 사용 후 HSK 1급 모의고사 평균 70점 이상
- 단어 장기 기억률 80% 이상 (1개월 후 재테스트)
- 발음 평가 점수 20% 이상 향상

### 6.2 사용자 참여
- 주 3회 이상 학습
- 평균 세션 시간 15분 이상
- 7일 연속 학습률 30% 이상
